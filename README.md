# Personal Portfolio (Localhost)

A lightweight personal portfolio website you can run locally and customize quickly.

## Run locally

```bash
python3 -m http.server 8080
```

Then open:

- http://localhost:8080

## Update with your own content

Edit `index.html` and replace placeholders in these sections:

- About Me
- Career Path
- Featured Work cards
- Video embed URL in the `<iframe src="...">`
- Contact details

## Files

- `index.html` → page content/structure
- `style.css` → styling/theme
- `script.js` → mobile menu + dynamic footer year
